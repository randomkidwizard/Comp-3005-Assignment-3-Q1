Steps:

1. Install library that will be used in the code: pip install psycopg2-binary

2. Change this line: "dbname='A3' user='postgres' host='localhost' password='postgres'"
	A3 - to the name of the database, postgres - to the name of the user in pgAdmin, 	password - to the password you use in postgres.

3. Now you can run the code.
